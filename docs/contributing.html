Kontribusi & Lisensi

Cara ikut membangun Jolt, dan syarat penggunaan lisensinya.

Cara Berkontribusi

Pull request sangat diterima! Pastikan kode mengikuti gaya Go dan sudah diuji sebelum dikirim.

bash 
# Clone fork
$ git clone https://github.com/IHx-cmyk/jolt
$ cd jolt
$ go mod tidy
$ go build -o jolt ./cmd/jolt
[info]Buka pull request ke branch main setelah perubahanmu lolos build dan teruji secara lokal.
